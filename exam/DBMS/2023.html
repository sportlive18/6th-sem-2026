✍️ Group-B (Short Answer – 5 Marks)
2. Armstrong’s Three Axioms

Reflexivity: If Y ⊆ X, then X → Y (e.g., (A,B) → A). Augmentation: If X→Y, then XZ→YZ (e.g., A→B ⇒ AC→BC). Transitivity: If X→Y and Y→Z, then X→Z (e.g., A→B, B→C ⇒ A→C).

3. Functional Dependency

Definition: X → Y means Y is functionally dependent on X; if two tuples have same X value, they must have same Y. Example: Student(ID, Name): ID → Name (one ID determines one name).

4. Lossless and Lossy Decomposition

Lossless: original relation reconstructed by join. Example: R(A,B,C) → R1(A,B), R2(B,C). Lossy: extra tuples created, cannot recover original. Example: R(A,B,C) → R1(A,B), R2(A,C).

5. Short Note: B+ Tree and B Tree

B-Tree: data in internal+leaf nodes, faster for small datasets, more height. B+ Tree: data only in leaves, internal nodes store keys, leaves linked → better range queries.

6. Metadata and Data Dictionary

Metadata: data about data (table names, column types, constraints). Data Dictionary: central repository storing metadata (structures, privileges, relationships).
